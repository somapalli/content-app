https://www.figma.com/make/51D5uGQSpXb0BN2kWoLqO0/Architecture-Diagram-for-Pricing-Flow?t=ZIoBdxoBcUzJw36O-1

Create an interactive architecture flow diagram showing the "Authorization, Settlement, Pricing Calculation, Ledger, and Merchant Payout Flow" with a comparison of Local Pricing Calculation vs Remote Pricing Calculation using Zafin Cloud.

The diagram should include 5 main sections:

1. Authorization Flow:

Gateway → Bank Transaction Processing Platform → Transaction Schema
Decision diamond: "Pricing Model?" with two paths:
Blended Pricing: Calculate fees immediately and record in Transaction Schema
Cost Plus Pricing: Wait for settlement data, continue to Settlement Flow
2. Settlement Flow:

Settlement File → Settlement Processing → Transaction Schema
Show fees recorded: Interchange, Network, Scheme, Processor fees
Then apply pricing logic and record fees/adjustments
3. Pricing Calculation Options (side-by-side comparison):

Option 1 - Local Pricing Calculation (Blue):

Transaction Platform → Local Pricing Engine → Transaction Schema
Key points: Lower latency, close to processing, better control, requires bank-owned rules and config sync
Option 2 - Remote Pricing using Zafin Cloud (Purple):

Transaction Platform → API Call → Zafin Cloud Pricing Engine → Response → Transaction Schema
Key points: Centralized rules, system of record, no duplicate logic, cloud dependency, network latency, requires resilience handling
4. Money Movement / Ledger (4 parallel flows):

Card Scheme Settlement → Operating Account
Operating Account → Revenue Account (Merchant Fees)
Operating Account → Expense Account (Issuer/Card Costs)
Operating Account → Merchant Payable Account (Net Settlement)
5. Merchant Payout:

Eligible Settled Transactions → Funding Aggregation → Merchant Payable Account → Merchant Bank Account
Note factors: Settlement rules, funding schedule, holds, reserves, adjustments, eligibility
Visual Design:

Use icons for: Gateway, Server, Database, Cloud, Files, Accounts, Dollar signs
Color code: Blue for Local pricing, Purple for Remote/Zafin pricing, Green for databases, Emerald for accounts
Include decision diamonds with rotation
Add arrows showing flow direction
Include a legend explaining all visual elements
Make it responsive and scrollable with clean, modern styling
